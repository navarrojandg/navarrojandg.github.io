JavaScript in the Browser

Given that JavaScript was born and raised in the browser, there are many different ways to execute JavaScript code within the browser. Below are are some of the ways to execute JavaScript within the web browser.

HTML's script tag

JavaScript is usually run through the use of the HTML script tag. By setting the src attribute for the script tag, the web page will download and execute the JavaScript file. JavaScript can also be executed by directly writing code on the HTML file itself. For example 

    <script>
        console.log('Hi! I was executed from the HTML tag!');
    </script>

Developer Tools

The 'console' tab in the developer console allows developers to quickly run JavaScript on the page that the developer tools were opened on.

URL / Bookmark

On most modern browsers JavaScript can be executed directly from the url by prefixing the code with 

javascript:
For example try copy and pasting this URL: 
javascript:alert('Hi! I was executed with the URL box!')

You can then save this URL as a bookmark to execute this snippet of JavaScript everytime you navigate to the bookmark.
