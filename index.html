Web Applications

Using NodeJS, a JavaScript runtime environment, applications that serve business logic through web servers are written primarily with JavaScript. NodeJS also allows developers to utilize JavaScript frameworks to develop single page applications that run only on the browser.

Art

Drawing images programmatically, like the house below can be done using JavaScript and the browser's Canvas API.

The code for this implementation is shown below

    var context = document.getElementById('canvas').getContext('2d');
    context.lineWidth = 8;

    // Wall
    context.strokeRect(75, 140, 150, 110);

    // Door
    context.fillRect(130, 190, 40, 60);

    // Roof
    context.beginPath();
    context.moveTo(50, 140);
    context.lineTo(150, 60);
    context.lineTo(250, 140);
    context.closePath();
    context.stroke();
